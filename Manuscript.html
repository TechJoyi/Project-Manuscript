1. Introduction

As an aspiring frontend developer and student of Computer Science at Yaba College of Technology, I developed a personal portfolio website to showcase my skills, learning progress, and future projects. This project is part of my efforts to build a solid foundation in web development and present myself professionally for internship opportunities in the tech field.

2. Project Objectives

  • To create a responsive website using core frontend technologies.
  • To present personal and academic information in a clean, user-friendly format.
  • To demonstrate a basic understanding of HTML, CSS, and JavaScript.
  • To deploy a real-world project and learn the basics of version control.
  • To use the website as a professional portfolio for internship applications.

3. Tools and Technologies Used

  • HTML5 – for structuring web content
  • CSS3 – for layout and styling
  • JavaScript (basic) – for interactivity
  • Visual Studio Code – development environment
  • Git & GitHub – for version control and project collaboration
  • Netlify / GitHub Pages – for website hosting

4. Project Overview

The personal portfolio website is designed to be responsive across devices. It includes:

  • Introduction Section – Brief overview of who I am
  • About Me – Educational background and career interests
  • Skills – Technologies I’m currently learning
  • Projects – Placeholder space for future work
  • Contact – Links to social platforms and email contact form

The project layout is mobile-first and uses media queries to adjust seamlessly to different screen sizes. A responsive navigation menu was built using JavaScript to enhance mobile usability.

5. Challenges Faced

  • Learning how to write clean, reusable CSS.
  • Understanding responsive design principles using Flexbox and media queries.
  • Initial difficulties deploying the site online using GitHub and Netlify.

6. Key Takeaways

  • Improved understanding of frontend structure and styling.
  • Gained practical experience in deploying real-world web projects.
  • Learned how to debug layout issues and structure responsive web pages.
  • Became familiar with using Git for version control.

7. Conclusion

This project represents an important milestone in my frontend development journey. It allowed me to apply the knowledge gained from self-study and coursework, while also preparing me professionally for internship roles. With continued learning, I aim to build more complex web applications and contribute meaningfully to any team I join.
